What is Phishing?

ANSWER: Phishing is a cybercrime in which a target or targets
are contacted by email, telephone or text message by someone
posing as a legitimate institution to lure individuals into providing
sensitive data such as personally identifiable information,
banking and credit card details, and passwords. The information
is then used to access important accounts and can result in
identity theft and financial loss.

Define error messages.

ANSWER: An error message is a message displayed to the user
by an operating system or application when an unexpected
condition happens. In most cases, error messages are displayed
with the help of dialog boxes by the operating system or
application. Error messages are required when user intervention
is needed or to pass important warnings.
